Detailed Description

Simple lock-free container mapping 32/64-bit keys to values. Fixed size, thread-safe. Limitation are:

  • Only maps 32/64 bit integers to 32/64 bit integers
  • All keys must be non-zero
  • Fixed maximum number of entries
  • Only operations are get/set
  • No true erase operation, only set to zero

Macro Definition Documentation

#define hashtable_t

Defined alias for a hash table storing values the size of a pointer, so 32 or 64 bit depending on architecture.

#define hashtable_allocate

Allocate storage for a pointer sized hash table of given size

#define hashtable_deallocate

Deallocate storage used by hash table

#define hashtable_set

Set stored value for the given key

#define hashtable_erase

Erase the value for a key by setting the value to zero. The key still holds a slot in the table.

#define hashtable_get

Get the value stored for the given key, or zero if no value stored

#define hashtable_size

Get number of stored keys with non-zero values. Walks the table so potentially slow.

#define hashtable_clear

Clear the entire table, resetting the stat to the state after initial allocation.
