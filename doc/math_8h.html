Detailed Description

Core math functionality, providing single entry points to common math functions across platforms and floating point notations used (32 or 64 bit real numbers).

Increment/decrement and wrap functions from http://cellperformance.beyond3d.com/articles/2006/07/increment-and-decrement-wrapping-values.html
