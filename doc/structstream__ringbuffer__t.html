Detailed Description

Stream interface for read/write to a ring buffer. This struct is also a stream_t (stream struct type declared at start of struct) and can be used in all functions operating on a stream_t. Read and write operation can be concurrent (one single thread reading, one single thread writing) as availability of data is flagged with semaphores. Multiple readers and/or writers are not supported. Stream is sequential.
