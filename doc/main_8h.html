Detailed Description

Main application entry points. The library implements the platform specific entry points into the application (like int main(...) for normal C library based applications) and handles the initial setup.

The foundation library then calls into these entry points to provide the application with a unified single entry point for each application lifetime event (initialize, run and finalize).

FOUNDATION_EXTERN int main_initialize ( void  )

Main initialization entry point. This must be implemented by the application. The expectation is that this function will call foundation_initialize to initialize the foundation library, and return an error code to indicate success or failure. If this method returns an error initialization will abort and the program will terminate.

Returns
0 for success, <0 for error
FOUNDATION_EXTERN int main_run ( void *  arg)

Main loop. This must be implementation by the application and should contain the main run loop. Once this function returns the application will terminate. The return code is used as the process exit code for normal termination. Will be called after main_initialize
