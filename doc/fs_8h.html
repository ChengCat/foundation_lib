Detailed Description

File system operations acting on the local operating system file system. All paths are either absolute paths, or relative paths to the current working directory. Absolute paths can optionally contain a file protocol declarator (file://)

Paths do not have to be zero terminated. Results from substring operations and other const string segments can be passed to all file functions.

Also provides an interface to monitor local file system changes through events on platforms supporting this.
